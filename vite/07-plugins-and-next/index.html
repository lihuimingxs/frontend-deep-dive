P7 · 插件 + 下一代横评

Plugin 是 bundler 的扩展机制；下一代 Rust bundler 是 2024+ 趋势。这一阶段两章把"工具链怎么演化"完整链条收尾。

章节数 2 主要来源 Rollup plugin API · Vite plugin docs · Unplugin · Turbopack/Rspack/Rolldown 官方 状态 ✅ 全部完成

本阶段覆盖

  • Rollup plugin hooks：resolveId / load / transform / buildStart / generateBundle 等
  • Vite 在 Rollup hooks 之上加的：config / configureServer / handleHotUpdate / transformIndexHtml
  • Plugin 顺序：pre / default / post 三档
  • Unplugin（Anthony Fu）：跨 Vite/Rollup/Webpack/esbuild/Rspack 的 plugin 抽象
  • 典型 plugin：vite-plugin-react / vue / pwa / unplugin-auto-import / unplugin-icons
  • Turbopack（Vercel）：Tobias Koppers 主导 + Rust + 增量编译 + Next.js 15+ stable
  • Rspack（字节跳动）：Rust + Webpack 兼容 API + "换上即加速"
  • Rolldown（Vite team）：Rust + Rollup 兼容 + 2026-Q2 stable 接班 Vite build
  • Bun build：Bun 自家 + Zig + JSC 集成（详见 Bun 主题）
  • Farm（中国社区）：Rust + Vite plugin 兼容 + 速度声称 5-10x Vite
  • 选型决策树：Next.js → Turbopack / Webpack 大项目迁移 → Rspack / Vite 升级 → Rolldown / Bun 项目 → Bun build / 新项目默认 → Vite / 库 → Rollup-Rolldown / 极致快 → esbuild / Bun build
  • 2026-2030 趋势：Webpack 持续退场 / Vite + Rolldown 双标准 / Rust bundlers 占 90%+ / Component Model 让 wasm bundler 集成
