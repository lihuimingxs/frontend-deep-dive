P1 · 构建工具是谁

不讲技术细节，只解决两个问题：构建工具 50 年怎么走到今天、同代人现在的格局。给后面所有"为什么这样设计"提供历史坐标。

章节数 2 主要来源 Evan You / Rich Harris / Tobias Koppers / Evan W 演讲博客 状态 ✅ 全部完成

本阶段覆盖

  • 1976 Make · "基于文件依赖图编译"模式起源
  • 2012 Grunt → 2013 Gulp · 早期 JS 任务化构建
  • 2013 Browserify → 2014 Webpack 1 · "什么都是模块"哲学诞生
  • 2015 Rollup · 专注 ESM + tree shaking
  • 2016 浏览器原生 ESM · 为 Vite 设计奠定基础
  • 2018 Parcel · 零配置挑战 Webpack
  • 2020 esbuild + Vite + SWC · 三连发动构建工具革命
  • 2022-2023 Webpack 主流退场 · Tobias 跳槽 Vercel 做 Turbopack
  • 2024+ Rust bundler 时代 · Turbopack / Rspack / Rolldown / Bun build
  • 同代人横评 · 10 个主流工具的全维度对比
