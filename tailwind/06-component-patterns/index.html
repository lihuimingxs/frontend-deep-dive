P6 · 组件模式

一个组件有多种变体（primary/secondary/outline/ghost）+ 多种尺寸（sm/md/lg）+ 用户传 className 想覆盖默认样式 —— 这是 Tailwind 在团队规模化下能不能不崩的关键。CVA + cn() + tailwind-merge + prettier-plugin-tailwindcss 是 2024+ 事实标准工具链。

章节数 1 主要来源 CVA docs / tailwind-merge docs / shadcn/ui 源码 / Joe Bell 公开演讲 状态 ✅ 全部完成

本阶段覆盖

  • CVA（class-variance-authority） · 类型安全的组件变体管理；Joe Bell 设计；shadcn/ui 推广
  • cn() helper · 来自 shadcn/ui，内部用 clsx + tailwind-merge
  • tailwind-merge · 解决 px-4 px-6 这种冲突 class，理解 Tailwind utility 优先级
  • prettier-plugin-tailwindcss · 自动按推荐顺序排列 class，团队 review 一致性
  • completed Button 模板 · variant + size + asChild 完整实现
  • VariantProps 类型推导 · 让 TS 编译时 catch 拼错的 variant
  • ESLint plugin 配合 · eslint-plugin-tailwindcss 帮 catch 错误 class
