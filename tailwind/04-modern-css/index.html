P4 · 现代 CSS 特性

Tailwind v4 的"押注现代 CSS"是 v3 → v4 重写的前提。这一章把 v4 用到的 7 个现代 CSS 特性讲透 —— cascade layers、@property、color-mix()、OKLCH、container queries、:has()、native nesting，以及它们怎么改变 Tailwind 的写法。

章节数 1 主要来源 tailwindcss.com/blog v4 + MDN CSS specs + Can I Use 兼容性数据 状态 ✅ 全部完成

本阶段覆盖

  • Cascade layers (@layer) · 2022 GA / cascade 优先级显式分层 / Tailwind v4 用它组织 base / components / utilities
  • @property · 注册自定义属性、有类型有默认值、可动画
  • color-mix() · 颜色插值 / opacity 修饰符（bg-blue-500/50）底层实现
  • OKLCH · perceptually uniform / wide gamut（P3 / Rec.2020）/ v4 默认调色板重做
  • Container Queries (@container) · 2023 GA / 组件级响应式 / Tailwind @sm: 容器前缀
  • :has() 选择器 · CSS 第一次有真正的 parent selector / Tailwind has-[...]: 变体
  • Native CSS Nesting · 2023-2024 浏览器主流支持 / Tailwind v4 不再依赖 PostCSS nested
  • Logical properties · v4.2 新 / inset-s / inset-e / inset-bs / inset-be 支持 RTL
