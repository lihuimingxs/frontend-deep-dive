P7 · 生态决策

Tailwind 主题的收束阶段。先讲 UI 库横评（Tailwind UI / Headless UI / Catalyst / shadcn/ui / DaisyUI 各自定位 + 何时用），再讲决策树（何时用 Tailwind / 何时不用 / Figma 对接 / 大型项目踩坑总结 / 2027 展望）。

章节数 2 主要来源 各 UI 库官方 + Adam Wathan / shadcn 公开访谈 + 真实生产案例 状态 ✅ 全部完成

本阶段覆盖

  • Tailwind UI · 商业组件库（299$ lifetime）/ HTML / React / Vue 三版本 / 设计参考
  • Headless UI · 开源 unstyled 组件（Dialog / Popover / Listbox）/ Tailwind Labs 出品
  • Catalyst · 商业 React UI kit / 比 Tailwind UI 更工程化
  • shadcn/ui · copy-paste 模式（CLI 复制源码到项目）/ Radix + Tailwind / 2026 React 事实标准
  • DaisyUI · Tailwind plugin 提供 semantic class（btn / card）/ 适合 Bootstrap 移民
  • NextUI / Mantine · 不基于 Tailwind 的 React UI 库（对照）
  • 何时用 Tailwind · React/Vue/Svelte/Astro + 团队接受 utility-first + 与 shadcn/ui 配合
  • 何时不用 · CMS / 邮件模板 / 极小项目 / 反 utility-first 文化
  • 设计系统对接 · Figma → Tokens Studio plugin / 多品牌 CSS variables / @theme 扩展
  • 大型项目踩坑 · 不要 @apply / safelist 动态 class / prettier-plugin 必装 / eslint-plugin-tailwindcss
  • 2027 展望 · Oxide 进一步成熟 / 现代 CSS 主流化 / shadcn/ui v3+ / AI 辅助 utility 生成
