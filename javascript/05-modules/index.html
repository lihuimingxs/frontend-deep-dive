P5 · 模块 & 元编程

模块是现代 JS 工程的骨架。本阶段把 ESM 规范（三阶段加载、live binding）、CJS 互操作、动态 import / tree-shaking / import maps 的构建器视角、polyfill 策略、eval 的安全边界、Web Workers 的多线程模型讲清楚 —— 覆盖从库作者到应用工程师的工具链全景。

章节数 8 / 8 ✅ 对应原站 Part 1.13 – 1.14 状态 P4 全部完成

本阶段覆盖

  • ESM 规范 —— 三阶段加载（parse/link/evaluate）、live binding、模块图、循环依赖、import.meta
  • CJS 互操作 —— function wrapper、循环 require、Node 双模系统、exports 字段、require(esm)
  • 动态 import —— 代码分割、webpack magic comments、modulepreload、import attributes、错误处理
  • Tree-shaking —— sideEffects 声明、pure annotations、桶文件陷阱、诊断方法
  • 模块解析 —— import maps、bare specifier、exports/imports 字段、条件导出、monorepo 路径别名
  • Polyfill 策略 —— core-js/preset-env 的 entry/usage 模式、baseline、differential serving、polyfill.io 风险
  • eval 安全边界 —— 直接 vs 间接 eval、new Function 隔离、CSP unsafe-eval、Trusted Types
  • Web Workers —— 三种 Worker 对比、Module Worker、Transferable 零拷贝、SAB + Atomics、Comlink RPC
