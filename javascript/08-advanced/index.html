P8 · 高级专题

把平时"需要时才查"的高级 API 体系化梳理一遍：二进制数据的完整模型（ArrayBuffer/TypedArray/DataView）、文件与表单（Blob/File/FormData）、流式处理（Streams API）、像素级渲染（Canvas/OffscreenCanvas）、流畅动画（rAF/WAAPI/FLIP）、浏览器原生组件化（Web Components）、正则引擎的回溯与 ReDoS、以及国际化数据格式化（Intl）。每一个主题都是 R&D 工程师在生产项目会撞到的坑。

章节数 8 章 · 约 18 万字 对应原站 Part 3.5 · Miscellaneous / Extra 状态 ✅ 已完成

章节列表

  1. 01 · ArrayBuffer / TypedArray / DataView

    原始字节容器、8 种 TypedArray、DataView 字节序、SharedArrayBuffer + Atomics、Transferable、与 WASM memory 互操作、Resizable ArrayBuffer。

  2. 02 · Blob / File / FormData

    Blob 不可变与懒存储、File 扩展、URL.createObjectURL 生命周期、FormData 自动 Content-Type、大文件分片上传 + 秒传。

  3. 03 · Streams API

    ReadableStream / WritableStream / TransformStream、pipeTo / pipeThrough、背压自动协调、tee 分流、BYOB、Compression/Decompression Streams。

  4. 04 · Canvas / OffscreenCanvas

    Canvas 2D 状态栈、HiDPI 缩放、ImageData 像素操作、OffscreenCanvas + Worker 渲染分离、ImageBitmap、跨源污染。

  5. 05 · Animation & requestAnimationFrame

    浏览器渲染管线、rAF vs setTimeout、属性成本分层（composite/paint/layout）、Web Animations API、FLIP、View Transitions、scroll-timeline。

  6. 06 · Web Components

    Custom Elements 生命周期、Shadow DOM 隔离、Slot 组合、Declarative Shadow DOM（SSR）、form-associated 元素、CSS :host / ::part / ::slotted、与 React 互操作。

  7. 07 · 正则表达式引擎

    NFA 回溯模型、贪婪 vs 惰性、灾难性回溯（ReDoS）防御、u / v flag 的 Unicode 语义、set notation、命名捕获组、lookahead/lookbehind、RE2 替代。

  8. 08 · Intl 国际化 API

    Intl.DateTimeFormat / NumberFormat / RelativeTimeFormat / Collator / PluralRules / ListFormat / Segmenter —— 替代 moment/dayjs/numeral 的原生方案。

阶段要点

  • 二进制栈：ArrayBuffer → TypedArray/DataView → Blob → File → Stream，层层抽象。
  • 渲染栈：Canvas → OffscreenCanvas + Worker → GPU（WebGL/WebGPU 指路）。
  • 动画：只动 transform/opacity、尊重 prefers-reduced-motion、帧预算 16.67ms。
  • 组件与模板：Web Components 是浏览器原生的跨框架组件模型。
  • 文本处理：正则要防 ReDoS，字符串要用 Intl.Segmenter 处理 emoji / CJK。
  • 国际化：Intl API 替代 90% 第三方库，始终带 locale 参数而非硬编码。
