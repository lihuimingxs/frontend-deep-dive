P2 · 语言基础

JavaScript 语法的基本面：类型、操作符、控制流、函数、对象、数组、集合。这一阶段的目标是把语言层的"形状"拼完整，后续阶段才能讨论更抽象的机制（作用域、原型、异步等）。

章节数 14 / 14 ✅ 对应原站 Part 1.1 – 1.14 状态 P1 全部完成

本阶段覆盖

  • 值与类型 —— 8 种类型、NaN-boxing、IEEE 754、UTF-16、装箱/拆箱、BigInt、Symbol
  • 操作与控制 —— ToPrimitive、== 规则、短路返回值、位运算、迭代协议、switch 严格相等
  • 函数与对象 —— 5 种函数形式差异、参数解构、属性描述符、getter/setter、freeze 层级
  • 集合数据 —— 数组方法全景 + Packed/Holey、字符串 Intl 家族、Map/Set/WeakMap/WeakSet
  • API 与协议 —— 数字/Math、Date/时区、JSON 三段处理、15 个 well-known symbols、迭代协议与 generator 双向通信
  • 工程基础 —— Error 层次与 cause 链、async 错误传播、全局捕获、严格模式
