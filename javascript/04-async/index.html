P4 · 异步 & 错误处理

JS 的并发模型是单线程 + Event Loop。本阶段把"异步"从语法糖层一路拆到规范层：任务队列、Promise 状态机、async/await 去糖、取消语义、并发控制、Node 特有行为。是写出健壮异步代码的唯一前置知识。

章节数 8 / 8 ✅ 对应原站 Part 1.10 – 1.12 状态 P3 全部完成

本阶段覆盖

  • Event Loop 模型 —— HTML 规范的 task queue、microtask queue、rendering cycle、task source 分类
  • 任务队列时序 —— queueMicrotask / MessageChannel / rAF / rIC / scheduler API 的决策表
  • Promise 状态机 —— PromiseState/Result 内部槽、resolve procedure 的 thenable 吸收、unhandled rejection 时机
  • Promise 组合器 —— all / allSettled / race / any 的规范语义与工程选择
  • async / await —— 去糖结果、错误传播、top-level await、forEach 陷阱
  • 取消协议 —— AbortController / AbortSignal.timeout / AbortSignal.any
  • 并发控制 —— 信号量、动态池、令牌桶、重试退避、批处理
  • Node 特性 + 错误收口 —— libuv 6 phase、process.nextTick、AsyncLocalStorage、graceful shutdown
