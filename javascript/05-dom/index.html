P5 · DOM & 事件

浏览器把 HTML 解析成 DOM 树后交给 JS —— DOM API 是 JS 操纵网页的唯一入口。本阶段把 DOM 接口家族、查询/修改/安全、事件传播机制、4 大 Observer、表单验证 API、页面加载时序讲完整。

章节数 8 / 8 ✅ 对应原站 Part 2 状态 P5 全部完成

本阶段覆盖

  • DOM 接口层次 —— Node/Element/Document、nodeType、children vs childNodes、attribute vs property、dataset
  • 查询与遍历 —— querySelector 与 getElementBy* 对比、live vs static、CSS 选择器性能、closest/matches、Range/TreeWalker
  • DOM 修改与安全 —— append/prepend/replaceChildren 家族、innerHTML XSS、Sanitizer API、template + DocumentFragment 批量插入
  • 事件传播 —— capture/target/bubble 三阶段、addEventListener 选项（passive/once/signal）、stopPropagation 的边界
  • 事件委托与自定义事件 —— closest 委托模式、CustomEvent + dispatchEvent、composed 跨越 shadow、EventTarget 作基类
  • Observer 家族 —— Mutation/Resize/Intersection/PerformanceObserver 的 API 与场景、rAF 时序
  • 表单 —— value/checked 陷阱、受控 vs 非受控、FormData、Constraint Validation API、ElementInternals 自定义表单控件
  • 资源加载 —— DOMContentLoaded/load/pageshow 时序、script defer/async/module、preload/modulepreload、Core Web Vitals
