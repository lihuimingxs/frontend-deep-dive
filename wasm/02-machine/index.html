P2 · 机器模型与格式

wasm 不只是"另一种字节码"——它的栈机器 + 二进制 module + 强类型 + 流式验证每一项都和性能 / 安全设计紧密绑定。这一阶段把它讲透。

章节数 2 主要来源 WebAssembly Core Spec · WABT · v8.dev/blog 状态 ✅ 全部完成

本阶段覆盖

  • wasm32 栈机器 ISA—— 指令在隐式操作数栈 push/pop，紧凑 + 易验证 + 引擎自由
  • 二进制 module 结构—— magic + version + 12 类 sections + LEB128 编码 + length prefix → streaming compilation
  • load-time validation—— wasm 安全模型核心，类型推导一次完成；失败 module 直接拒绝
  • wat 文本格式—— S-expression，与二进制 1:1 对应；调试 / 教学用
  • 类型系统—— 数值（i32/i64/f32/f64）+ 向量（v128）+ 引用（funcref/externref）；multi-value 多返回值
  • 结构化控制流—— block/loop/if/br/br_if，没有任意 goto；让验证 + 编译器优化都可行
