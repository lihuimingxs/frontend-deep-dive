P4 · §6 数据类型与值

§6 把"类型"分为两大类：Language Types（用户可见的 8 种）和 Specification Types（规范内部：Reference Record / Completion Record / Property Descriptor 等）。Specification Types 不是 JS 值——你写不出 typeof x === 'CompletionRecord'——但它们是规范文本里反复出现的"参数 / 返回值 / 中间结果"，不理解它们就读不懂 (0, obj.x)() 为什么丢 this。

章节数 1 / 1 ✅ 对应 ECMA-262 §6 ECMAScript Data Types and Values 状态 完成

本阶段覆盖

  • 8 个 Language Types —— Undefined / Null / Boolean / String / Number / BigInt / Symbol / Object
  • Specification Types —— Reference Record / Completion Record / Property Descriptor / Environment Record / Abstract Closure / Data Block
  • Number 内部 —— IEEE 754 双精度、+0 vs -0、NaN 不等于自己、Number.EPSILON
  • String 内部 —— UTF-16 code unit 序列（不是 code point！）、surrogate pair、Intl.Segmenter
  • Symbol & Well-known Symbol —— 协议钩子（@@iterator / @@toPrimitive 等）
