P6 · §10 内部槽与方法

§10 是"对象本质"的规范层定义：每个对象有 11 个 essential internal methods（[[GetPrototypeOf]] / [[Get]] / [[Set]] / [[Delete]] 等）+ 若干内部槽。普通对象的 internal methods 走"Ordinary 默认实现"，7 种 Exotic Object（Array、String、Arguments、Module Namespace、Proxy、Bound、TypedArray）覆盖部分方法实现特殊行为。Proxy 的本质 = "替换全部 11 个 internal methods 为 trap"——理解这一层，Proxy 立刻变得朴素直白。

章节数 1 / 1 ✅ 对应 ECMA-262 §10 Ordinary and Exotic Objects Behaviours 状态 完成

本阶段覆盖

  • 11 个 essential internal methods —— [[GetPrototypeOf]] / [[Get]] / [[Set]] / [[Delete]] / [[OwnPropertyKeys]] 等
  • Ordinary Object —— 默认实现（OrdinaryGet 等），属性表 + 原型链委托
  • 7 种 Exotic Object —— Array（length 自动同步）/ String（index → UTF-16 unit）/ Arguments / Module Namespace（frozen）/ Proxy / Bound function / Integer-Indexed (TypedArray)
  • 函数特有 —— [[Construct]] / [[Call]]
  • PrivateElement / PrivateName —— class #field 的规范层
