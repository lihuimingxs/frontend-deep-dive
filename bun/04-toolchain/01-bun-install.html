bun install · binary lockfile · Catalogs · hard link · workspaces

包管理器章。Bun 主题里最容易让用户切换的工具——通常工程师从 bun install 开始用 Bun，再考虑 runtime / build。关键洞察：hard link vs pnpm symlink 是核心取舍——Bun 用 hard link 兼容性好，但 phantom dep 检测严格度不如 pnpm。

阅读时长 ~30 分钟 前置 pnpm 经验有助 主要来源 bun.com/docs/install · Bun 1.2 binary lockfile blog
