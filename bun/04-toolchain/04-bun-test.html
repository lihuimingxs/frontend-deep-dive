bun test · Jest 兼容 · snapshot · 与 Vitest/node:test 横评

测试 runner 章。bun test 是 Jest 兼容 + 启动快的方案——和 Vitest / node:test 三方混战。关键洞察：bun test 定位是「已用 Jest 想加速」；Vitest 护城河是 DOM 测试 + UI dashboard；node:test 护城河是零依赖 + 与 Node 同生命周期——三者各有不可替代的场景。

阅读时长 ~30 分钟 前置 Node P6.1 node:test 主要来源 bun.com/docs/test · Vitest docs · Jest docs
