Bun.serve / Bun.file / Bun.sql / Bun.password / Bun.glob

把 Bun 的「内置 API 套」系统讲一遍——它们的设计共性是什么、为什么比 Node + npm 等价物快。关键洞察：内置不是为了「少装包」，是为了「零拷贝路径」——Bun.serve(req).write(Bun.file()) 走 sendfile 系统调用，绕过用户态 buffer，这种优化第三方包永远做不到。

阅读时长 ~30 分钟 前置 Node P4.2 HTTP + Node P4.3 fs 主要来源 bun.com/docs/api · Bun 1.2 release notes
