Node API 兼容矩阵（2026）+ N-API + 何时 fallback

Bun 的「Node 兼容」是它最大卖点，但兼容到哪一步是 production 决策的关键。这一章讲清 2026 年的兼容现状 + 何时仍需 fallback。关键洞察：兼容层不是「copy Node 实现」——Bun 用 Zig 重写了 fs/http/crypto，表面 API 一样，行为对齐，性能更好；但角落 case 仍可能差异，必须查 compat:nodejs Issues。

阅读时长 ~30 分钟 前置 P3.1 + Node P3 / P4 / P5 主要来源 github.com/oven-sh/bun/issues · bun.com/docs/runtime/nodejs-apis
