JSC 4 层流水线 + Structure + 启动快的真正原因

JSC 视角。讲清 JSC 的设计选择、与 V8 的差异、对 Bun 启动速度 / 内存的影响。关键洞察：Bun 选 JSC 不是因为「全方面优于 V8」——V8 长期跑 hot 代码上 TurboFan 略好；JSC 优势是冷启动 + 内存占用 + 嵌入开销，正是服务端 / CLI / 边缘场景核心。

阅读时长 ~30 分钟 前置 V8 P2 编译流水线 + V8 P3.1 Hidden Class 主要来源 webkit.org/blog/javascriptcore · WebKit 源码 · Bun 团队博客
