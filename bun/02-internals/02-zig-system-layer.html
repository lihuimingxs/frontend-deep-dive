Zig comptime / uSockets / 为什么不选 Rust

Zig 视角。讲清 Zig 在 Bun 里扮演什么角色、为什么不是 Rust / C++、Zig + uSockets 怎么替代 libuv。关键洞察：Zig 不强制安全反而是 Bun 优势——comptime 让「硬编码」不再是反模式（专用代码免费），uSockets 在 Linux 直接 io_uring 跳过 epoll 包装。

阅读时长 ~30 分钟 前置 P2.1 + Node P2.1 libuv 主要来源 ziglang.org · uNetworking/uSockets · Bun 源码
