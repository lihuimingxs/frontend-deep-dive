P1 · 测试工具是谁

不讲技术细节，只解决两个问题：测试工具 15 年怎么走到今天、同代人现在的格局。给后面所有"为什么这样设计"提供历史坐标。

章节数 2 主要来源 Kent C. Dodds / Anthony Fu / Andrey Lushnikov 公开文章 + 各家官方公告 状态 ✅ 全部完成

本阶段覆盖

  • 2011 Mocha · TJ Holowaychuk 第一个流行 Node runner
  • 2010-2014 Jasmine · BDD 描述语法（describe / it / expect）成 API 模板
  • 2014 Jest · Facebook 出品；自带 mock/snapshot/并行；React 默认
  • 2017 Cypress · 第一个全图形界面 e2e；time-travel debugging
  • 2017 Puppeteer → 2020 Playwright · CDP 直驱；多浏览器接班
  • 2021 Vitest · Anthony Fu；Vite 生态原生 ESM runner
  • 2022 Enzyme 死亡 · React 18 不兼容 + 维护停滞
  • 2023-09 Bun test · Jest 兼容 + Bun 速度
  • 2024 node:test 进 LTS · 内置不需装；轻量但功能不全
  • 2024 MSW 2.0 · Service Worker + Node + WebSocket 三模一统
  • 同代人横评 · 8 家 runner + Cypress/Playwright + Testing Library 全家族
