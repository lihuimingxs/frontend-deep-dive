P2 · 测试心智

这一阶段不讲工具，讲心智。理解"金字塔 vs 奖杯"和"测行为不测实现"这两个原则后，你才知道为什么 Vitest / Testing Library / Playwright 这套设计是合理的——后面所有工具知识才有锚点。
