P3 · UI hooks + Generative UI

useChat 是 Vercel AI SDK 最重要的 hook。这一阶段把 useChat / useCompletion / useObject 全谱（v5+ UIMessage 心智 + parts 类型）和 Generative UI（streamUI 在 Server Action 直接渲染 React 元素 + RSC 集成 + 与 React 19 use API 协作）讲清。

章节数 2 主要来源 ai-sdk.dev/docs/ai-sdk-ui + ai-sdk.dev/docs/ai-sdk-rsc + Vercel useChat 实战博文 + React 19 use API RFC + Next.js RSC 文档 状态 ✅ 全部完成

本阶段覆盖

  • useChat 核心 · messages / input / handleSubmit / status / sendMessage 完整 API
  • UIMessage parts 类型 · text / tool-{name} / data-{custom} / source / step-start 等
  • convertToModelMessages · UI ↔ Model 边界转换
  • useCompletion · 单次补全 vs useChat 多轮
  • useObject · 流式生成 Zod 结构化对象（form / 表格 UI）
  • streamUI（ai/rsc）· LLM 决定渲染什么 React 组件 → server 直接渲染
  • RSC 集成 · 通过 RSC payload 边收边渲染（链回 React + Next.js 主题）
  • 与 React 19 use API 协作 · server 创建 promise + client use 消费
  • useChat vs streamUI 选择 · 简单 chat vs 丰富 UI 渲染
