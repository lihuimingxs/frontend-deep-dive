P5 · Claude Agent SDK + MCP

Claude Agent SDK 是 Anthropic 给"构建生产 AI agents"的官方库，与 Claude Code 共享 agent loop。这一阶段把它的设计哲学（agent loop / subagents / hooks / sessions）+ 与 Vercel 两种范式 8 维对比 + MCP 协议在两边的位置（Claude SDK 中心地位 / Vercel 仍 experimental）讲清。

章节数 2 主要来源 code.claude.com/docs/en/agent-sdk + github.com/anthropics/claude-agent-sdk-{python,typescript} + modelcontextprotocol.io + Anthropic engineering blog + Cursor / VS Code MCP 实现 状态 ✅ 全部完成
