INP 替代 FID + Long Tasks + scheduler.yield/postTask

从用户输入到响应完成的性能。讲透 INP（2024 起替代 FID 的核心 Web Vital ≤ 200ms）+ Long Tasks（> 50ms 阻塞主线程）+ PerformanceObserver API + scheduler.yield()（Chrome 129+ 让出主线程）+ scheduler.postTask（精细调度 user-blocking/visible/background）+ Event 处理链 + isInputPending + requestIdleCallback。

阅读时长 ~30 分钟 前置 P4.1 主要来源 web.dev/inp · web.dev/optimize-long-tasks · scheduler API
