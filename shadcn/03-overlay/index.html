P3 · 浮层组件深度

浮层组件不是 "CSS position: fixed" 这么简单 —— Floating UI 智能定位、Portal 解决 z-index 战争、focus trap 让 Tab 在 Dialog 内循环、scroll lock 处理 iOS 橡皮筋。这一章把 Dialog / Popover / Dropdown / Tooltip / HoverCard / ContextMenu 全谱讲透。

章节数 1 主要来源 radix-ui.com docs + Floating UI docs + WAI-ARIA APG 状态 ✅ 全部完成

本阶段覆盖

  • 浮层组件家族 · Dialog / Popover / DropdownMenu / Tooltip / HoverCard / ContextMenu
  • Floating UI 定位 · 自动避开视口边缘 / 翻转方向 / collision detection / arrow 定位
  • Portal · 把浮层渲染到 document.body 末尾，解决 z-index 战争
  • Focus Trap · Dialog 打开 Tab 在内循环；关闭 focus 回触发元素
  • Scroll Lock · Dialog 打开禁止 body scroll；处理 iOS 橡皮筋 + scrollbar gutter
  • ESC + 点外关闭 · 全局 keydown / click 监听；嵌套时只关最上层
  • 浏览器原生 popover API · HTML <dialog> / popover attribute；Radix 暂未全切
  • 动画 · data-state="open"/"closed" + Tailwind animate-in/out
  • Tooltip 与 disabled button · disabled 不触发 hover，要包 div
  • 嵌套 Dialog / Modal 之上的 Popover · z-index 与 focus 处理
