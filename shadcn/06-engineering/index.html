P6 · 工程实践

shadcn 项目最常踩坑的两块：Form 完整工作流（react-hook-form + Zod + shadcn Form 三方协同 + Server Actions 集成）、Theme + 暗色 + 多品牌（CSS variables + next-themes + theme builder + Tailwind v4 集成）。

章节数 2 主要来源 react-hook-form docs + zod docs + shadcn Form docs + next-themes 状态 ✅ 全部完成

本阶段覆盖

  • 三方角色 · react-hook-form（state）+ Zod（schema）+ shadcn Form（wrapper）
  • shadcn Form 组件链 · Form / FormField / FormItem / FormLabel / FormControl / FormDescription / FormMessage
  • 典型 flow · Zod schema → z.infer → useForm + zodResolver → render → onSubmit
  • 与 Server Actions 配合 · useFormStatus / useActionState / 双层 Zod 校验
  • 动态字段 useFieldArray · 数组字段（动态 todo / multi-step）
  • shadcn token 系统 · --background / --primary / --muted 等
  • 暗色 next-themes · class 策略 + localStorage 持久化
  • shadcn theme builder · ui.shadcn.com/themes 可视化选色
  • 多品牌 white-label · CSS variables 在 .theme-x 切换
  • Tailwind v4 @theme 集成 · CSS-first 配置
