P1 · shadcn + Radix 是谁

不讲组件 API 细节，只解决两个问题：Radix 怎么从 2018 年小众 headless 库走到 shadcn 推广 + 2026 双引擎事实标准、它在 React UI 生态站哪。给后面所有"为什么这样设计"提供历史坐标。

章节数 2 主要来源 ui.shadcn.com/docs/changelog + radix-ui.com releases + shadcn 公开访谈 状态 ✅ 全部完成

本阶段覆盖

  • 2018-2020 Radix 起步 · Pedro Duarte / Vlad Moroz 在 Modulz 启动；目标 a11y-first headless 组件
  • 2020-12 Radix Primitives v1 · Dialog / Popover / Dropdown 等核心 primitives 开源
  • 2022 Radix Themes · Modulz 团队推出 pre-styled 组件库（与 Primitives 并行产品线）
  • 2023-03 shadcn 公开 · Hassan El Mghari 把"Radix + Tailwind + CVA 模板"公开为 CLI
  • 2024 现象级 · Vercel / Linear / Resend 等公开使用；GitHub stars 50k+
  • 2024-09 Radix 加入 WorkOS · Modulz 团队整体加入 WorkOS（保证长期维护）
  • 2026-02 双引擎 · shadcn 加 Base UI 支持；Radix Primitives 出统一包 radix-ui + tree-shaking
  • 2026-02 1300+ blocks · Registry 扩展（dashboard / form / login / sidebar 全套，Radix + Base UI 双变体）
  • 2026-03 CLI v4 · Visual Project Builder + 多框架 scaffold
  • 同代人横评 · Headless（Headless UI / Ark UI / Park UI）vs Styled（Mantine / NextUI / MUI / Chakra）vs Hybrid（shadcn / Catalyst）三阵营
  • 4 个核心设计赌注 · Headless + 标准 a11y / Copy-paste 不 npm install / CVA + cn 标准模式 / 双引擎可选
