P4 · 表单组件深度

表单 widget 比浮层更难（状态多、a11y 严格、与表单系统集成）。这一章把 Checkbox / RadioGroup / Switch / Slider / Select / Combobox 全讲透，重点 Combobox（最难的 widget）+ shadcn 用 cmdk 而不是 Radix 的工程权衡。

章节数 1 主要来源 radix-ui.com docs + cmdk docs + WAI-ARIA APG Combobox/Listbox 状态 ✅ 全部完成

本阶段覆盖

  • 简单控件 · Checkbox / RadioGroup / Switch / Slider / Toggle / ToggleGroup
  • Checkbox · indeterminate state、aria-checked
  • Slider · 单值/范围、键盘 ←→↑↓ Home End、aria-valuemin/max/now
  • Select · 下拉选择、单选、与原生 <select> 类似 a11y
  • Combobox · 最复杂的 widget（输入 + 自动完成 + 选项 + 异步 + 多选）
  • shadcn Combobox 用 cmdk · Radix 没自带 Combobox primitive，cmdk 由 Linear 团队开源提供
  • aria-activedescendant · 不真 focus 选项，避免输入框失焦
  • Listbox a11y · 选项列表本身的 a11y 标准
  • Multi-select Combobox · 选中标签 + 清除 + 键盘删除
  • 异步加载 · debounce + cache + loading state
