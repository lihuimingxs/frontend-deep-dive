P2 · HTTP/1.1 基线

HTTP/1.1 在 2026 年仍占 web 流量约 25-30%，是 fallback 主线。这一阶段把 RFC 9110-9112 修订视角下的 1.1 真相讲清：keep-alive / chunked / Range / Host / pipelining 怎么工作，6 connection limit 从哪来，应用层 HOL blocking 是后面所有协议演化的根本动机。

章节数 1 主要来源 RFC 9110 (HTTP semantics) + RFC 9111 (caching) + RFC 9112 (HTTP/1.1 wire) + Robin Marx 的 HOL blocking 研究 + Web Almanac 2024-2025 协议层流量数据 状态 ✅ 全部完成

本阶段覆盖

  • RFC 9110-9112 修订视角 · 2022 年 HTTP semantics 大重构后 1.1 真正的 wire format 是什么
  • 5 个核心机制 · keep-alive / chunked transfer / Range + 206 / Host header / pipelining
  • 3 个天花板 · 应用层 HOL blocking · 6 connection limit · headers 重复发送
  • 2026 仍活着的根因 · curl 一行调试 · 内网工具链 · CDN ↔ origin 段 · HTTP/2 共享 1.1 语义
