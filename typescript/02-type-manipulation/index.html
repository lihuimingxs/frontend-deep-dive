Part 2 · 类型操作

这是 TS 真正与 JS 拉开差距的部分。Part 1 的内容大多在"如何描述静态形状"，Part 2 进入"如何用类型计算类型"——把类型系统当成一个图灵完备的小型语言，从已有类型推出新类型。所有读 TS 库源码（react / zod / drizzle / @types/express）的能力都在这 6 章。

章节数 6 章 对应官方 Handbook v2 · Type Manipulation 前置 Part 1 全部

本 Part 解决什么问题

  • 怎么写一个真正"类型安全的工具函数" —— 07 Generics 让函数的输入输出类型联动
  • 怎么从一个对象类型反向取出键名 / 取出值类型 —— 08 keyof / typeof / Indexed Access
  • 怎么写"参数是 string 时返回 X，是 number 时返回 Y"的类型 —— 09 Conditional Types + infer
  • 怎么写一次循环把对象的所有字段变成 readonly / optional / rename —— 10 Mapped Types
  • TS 标准库里 Partial / Pick / Omit 是怎么实现的 —— 11 Utility Types 源码解读
  • 怎么用 satisfies 让配置对象既被检查又保留精确推断 —— 12 satisfies + 综合实战

章节清单

序号 章节 核心收获
07 Generics 延迟绑定、约束、双类型参数互相约束、const T（5.0）、NoInfer（5.4）
08 keyof / typeof / Indexed Access 类型查询三件套，把"取属性 / 反推类型"操作类型化
09 Conditional Types & infer 分布式行为、infer 模式匹配、tail recursion；理解 ReturnType / Awaited / Exclude 实现
10 Mapped Types & Template Literal modifier 加减、key remapping as、笛卡尔积、Capitalize 等 intrinsic
11 Utility Types 22+ 标准 utility 源码逐个解读，从实现学技巧
12 satisfies 与组合艺术 : T / as T / satisfies T 三者本质差异、branded type、组合三剑客做 DSL

本 Part 的难度曲线

从 07 到 12 难度陡升再回落：

  • 07 Generics —— 中等。已知概念稍作延展
  • 08 keyof / typeof —— 中等。引入"类型值反推"
  • 09 Conditional Types ⚠️ —— 本 Part 最难。distributive 行为 + infer 模式匹配，需要花最长时间
  • 10 Mapped Types —— 中等偏难。配合 09 一起读
  • 11 Utility Types —— 中等。有了 09/10 基础后回归"应用"
  • 12 satisfies —— 中等。综合实战

建议阅读顺序

  1. 顺序读 07 → 08（基础工具）
  2. 09 慢读，多写代码验证。distributive 那一节读两遍
  3. 10 紧跟 09（很多 mapped type 内部用 conditional）
  4. 11 把 lib.es5.d.ts 翻出来对照看 — 最有冲击力
  5. 12 收尾综合实战
