P5 · 嵌入与诊断

V8 不是命令行工具，是个 C++ 库。它的真正用户是 Chrome / Node / Deno / Workers / Electron。这一阶段讲怎么把 V8 嵌入进 C++ 程序、怎么用 --trace-* 系列诊断生产代码、以及 V8 Sandbox 这个 2024 重大安全里程碑。

章节数 2 主要来源 v8.dev/docs/embed · v8.dev/blog/sandbox · Node embedding API 状态 ✅ 全部完成

本阶段覆盖

  • Embedder API 核心对象层级 —— Isolate（≈ Agent）/ Context（≈ Realm）/ HandleScope / Persistent / TracedReference
  • Snapshot —— Startup Snapshot + Custom Snapshot；Workers < 5ms 冷启动核心 trick
  • Promise / Microtask Hooks —— Embedder 决定 microtask drain 时机；Node async_hooks 的下游
  • C++ ↔ JS 互调 —— FunctionTemplate / ObjectTemplate / N-API
  • 诊断工具全谱 —— --trace-deopt / --trace-ic / --trace-turbo / --trace-gc / d8 / Turbolizer / Inspector Protocol
  • V8 Sandbox（2024 默认开启） —— pointer cage + external pointer table + code pointer table + trusted space
