P2 · 渲染心智模型

写 React 组件需要建立的核心心智模型。讲清 React 元素是对象不是 DOM、JSX 是 JS 表达式不是模板、组件是函数（class 已老）、Props 设计与 ref as prop（19）、State 触发 render 的批处理与函数式更新、合成事件系统、受控 vs 非受控表单。这是写业务 React 的必读阶段。

章节数 5 / 5 ✅ 主要来源 react.dev Reference · Babel/SWC JSX 转换 状态 ✅ 全部完成

本阶段覆盖

  • React 元素 —— 对象 {type, props, key, ref} 不是 DOM；JSX 编译产物
  • JSX runtime —— 新 runtime（17+） vs classic；TypeScript 中的 JSX；命名规则（小写 → DOM、大写 → 组件）
  • 组件与 Props —— function component（默认）/ class（已老但仍维护）；Props 设计原则；children；ref as prop（19）
  • State 基础 —— useState 用法层；setState 触发 render 的语义；批处理（自动 batching）；函数式更新；状态提升
  • 条件渲染 / 列表 / 事件 / 表单 —— 条件渲染模式（&&、三目、变量）；列表 + key 设计；合成事件系统；受控 vs 非受控表单
