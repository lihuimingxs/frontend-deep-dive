P6 · Server Components 与 Actions

React 19 时代的核心新特性。Server Components（RSC，2024 稳定）+ Actions（表单 progressive enhancement）+ use API（在 render 中读 Promise + Context）+ 现代杂项（ref as prop / Context 简化 / document metadata）。这是 React 19 区别于 React 18 的最大差异面。

章节数 4 / 4 ✅ 主要来源 RFC #188（RSC）· RFC #229（Actions）· react.dev/blog/2024 状态 ✅ 全部完成

本阶段覆盖

  • Server Components —— 心智（服务器状态 + 客户端状态两层模型）/ 'use client' 边界 / 流式渲染 / 与传统 SSR 的区别 / RFC #188 设计动机
  • Actions —— useActionState / useFormStatus / useOptimistic / 表单 progressive enhancement / 与 Server Actions 配合 / RFC #229
  • use API —— 在 render 中读 Promise（搭配 Suspense）+ 读 Context / 与 useContext 的关系 / 何时用
  • 现代杂项 —— ref as prop（forwardRef 19 退役）/ <Context> as provider / document metadata / 新 hydration 错误
